# PRÁCTICA 1

Esta práctica consistió en crear una página HTML muy básica con estilos y un archivo de JavaScript para ejecutar algunas funciones.

---

## Cómo ver la práctica

Para ver la práctica, solo hay que descargar los archivos y ejecutar **Live Server** desde Visual Studio Code.

Además, hay que abrir el **inspector de Chrome** para poder ver la salida de las funciones de JavaScript.

Ejemplo de salida en consola:

```javascript
console.log("Resta:", res);
```
